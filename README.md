# Road Mobile

A Flutter-based community navigation app for Malaysia with real-time traffic updates, district forums, and interactive map integration.

## Features

- Regional forums for Malaysian districts
- Interactive map with clickable district markers
- Community-driven posts with categories (accidents, traffic jams, roadblocks, road closures, potholes, weather, construction)
- Real-time comments and updates via Firebase Firestore
- Photo uploads for posts
- Anonymous or named posting options
- iOS-style Cupertino UI design

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/road-mobile.git

# Navigate to project directory
cd road-mobile

# Install dependencies
flutter pub get

# Configure Firebase
flutterfire configure

# Run the app
flutter run
```
